🛠️ معلومات الأدوات المتاحة

        
الأدوات الافتراضية المدمجة في الوكيل:


        

          

            
🔍

            
search

            
البحث على الويب عبر DuckDuckGo / Google

          

          

            
🧭

            
navigate

            
الانتقال إلى عنوان URL محدد

          

          

            
👆

            
click

            
النقر على عنصر بواسطة الفهرس

          

          

            
⌨️

            
input

            
إدخال نص في حقل نموذج

          

          

            
📜

            
scroll

            
التمرير لأعلى أو لأسفل في الصفحة

          

          

            
📋

            
extract

            
استخراج بيانات من صفحة الويب بالذكاء الاصطناعي

          

          

            
📷

            
screenshot

            
التقاط لقطة شاشة للتحقق البصري

          

          

            
🗂️

            
switch / close

            
إدارة علامات تبويب المتصفح

          

          

            
📂

            
write_file / read_file

            
قراءة وكتابة الملفات المحلية

          

          

            
🏁

            
done

            
إنهاء المهمة وإرجاع النتيجة

          

          

            
⏳

            
wait

            
الانتظار لعدد محدد من الثواني

          

          

            
↩️

            
go_back

            
العودة إلى الصفحة السابقة في التاريخ

          

        


        

          💡 نصيحة:
          يمكنك إضافة أدوات مخصصة عبر الكود البرمجي باستخدام @tools.action(...).
          راجع التوثيق لمزيد من التفاصيل.
